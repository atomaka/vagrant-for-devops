##  Usage

* Primary commands
  * ```vagrant up```
  * ```vagrant ssh```
  * ```vagrant destroy```
* Others
  * ```vagrant provision```
  * ```vagrant halt```
  * ```vagrant suspend```
  * ```vagrant resume```
  * ```vagrant push```

note:
    Primary: mandatory; stuff I always use

    Others: commands I sometimes use

    Push: recent command for doing deploys; can deploy to heroku
